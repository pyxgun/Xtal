# Xtal
## What Xtal is
Xtal /krístəl/ is a small container runtime written in Nim.

## Features
- pivot root
- namespaces(IPC, Network, Mount, PID, User, UTS)
- memory & cpu limits
- pull image from Docker Hub

## Installation
```
  git clone https://github.com/pyxgun/xtal.git
  cd xtal
  nimble build
```
